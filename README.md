# 🏓 Ping Pong Master

O **Ping Pong Master** é um site desenvolvido com o objetivo de simular uma loja virtual de bonecos criados com o auxílio de Inteligência Artificial (Nano Banana).

Este projeto foi proposto pelo professor da disciplina de **Desenvolvimento de Aplicações Dinâmicas**, com a finalidade de praticarmos a construção completa de um site utilizando HTML e CSS, preparando a base para futuramente adicionar interatividade com JavaScript.

---

## 🚀 Objetivo do Projeto

Desenvolver um site de vendas completo, aplicando conceitos de:

- Estruturação semântica com HTML  
- Estilização e organização visual com CSS  
- Simulação de um ambiente de e-commerce  
- Introdução à interatividade com JavaScript  

O projeto foi pensado como base para evolução futura, permitindo a implementação de animações, efeitos visuais e funcionalidades mais dinâmicas conforme o avanço na disciplina.

---

## 🛠 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript (utilizado para implementação de um slider de imagens)  
- Inteligência Artificial (Nano Banana) para criação dos bonecos  

---

## 📌 Funcionalidades da Primeira Versão

- ✔ Estrutura completa do site em HTML  
- ✔ Estilização completa utilizando CSS  
- ✔ Implementação de um slider em JavaScript  
  - Alterna entre a imagem antiga e a nova versão do boneco  
- ✔ Layout organizado simulando um site de vendas
